快速入门指南将会向你展示最快速的安装、配置、启动方式。

<Note>
  **前期准备**:
  在开始之前，你至少需要有一个 Home Assistant 实例，并已添加至 Home Assistant 官方服务端。
</Note>

## 环境准备

请先安装好 node 与 pnpm。

<Note>如果你已经准备好 node & pnpm 可以跳过此步骤</Note>

<Steps>
  <Step title="安装Node">
    请前往 [Node官方安装地址](https://nodejs.org/en/download) 进行安装，并验证安装是否成功。

    以下为 node:22(LTS) + pnpm 的安装方式
    <CodeGroup>
      ```bash macos / linux
      # 下载并安装 nvm:
      curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash

      # 重启shell
      \. "$HOME/.nvm/nvm.sh"

      # 下载并安装 Node.js:
      nvm install 22

      # 验证 Node.js 版本:
      node -v # 应该打印出 "v22.19.0".

      # 下载并安装 pnpm:
      corepack enable pnpm

      # 验证 pnpm 版本:
      pnpm -v
      ```

      ```bash windows
      # 下载并安装 Chocolatey:
      powershell -c "irm https://community.chocolatey.org/install.ps1|iex"

      # 下载并安装 Node.js:
      choco install nodejs --version="22.19.0"

      # 验证 Node.js 版本:
      node -v # 应该打印出 "v22.19.0".

      # 下载并安装 pnpm:
      corepack enable pnpm

      # 验证 pnpm 版本:
      pnpm -v
      ```
    </CodeGroup>

  </Step>

  <Step title="将以下MCP配置信息添加到你的MCP客户端中">
    ```json
    {
      "mcpServers": {
        "home-mcp": {
          "command": "npx",
          "args": ["home-mcp"],
          "env": {
            "HASS_TOKEN": "<你的Home Assistant长期访问令牌>",
            "HASS_URL": "http://localhost:8123"
          }
        }
      }
    }
    ```
  </Step>
</Steps>
